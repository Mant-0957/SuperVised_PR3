#  Supervised Machine Learning Project

A complete **Supervised Machine Learning** project demonstrating the end-to-end workflow of building predictive models using Python and Scikit-learn. This project covers data preprocessing, exploratory data analysis, model training, hyperparameter tuning, and performance evaluation.

---

##  Project Overview

The goal of this project is to build and compare multiple supervised machine learning models to predict outcomes from a structured dataset. The notebook follows a real-world machine learning pipeline from data loading to final model evaluation.

---

##  Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Train-Test Split
* Model Building
* Hyperparameter Tuning using GridSearchCV
* Model Performance Evaluation
* Prediction on New Data
* Easy-to-understand code with explanations

##  Installation

Clone the repository

```bash
git clone https://github.com/Mant-0957/SuperVised_PR3.git
```

Move into the project directory

```bash
cd SuperVised_PR3
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
